#Entire Conatiner
mt-20 → Adds top margin (5rem = 80px).
md:mt-28 → On medium+ screens, increase top margin to 7rem (112px).
md:w-2/3 → On medium+ screens, width is 2/3 of the parent container.
md:self-start → On medium+ screens, aligns this div to the start (top/left depending on flex direction of parent).
flex flex-col → Makes the container a flexbox with vertical (column) direction.
items-center → Centers items horizontally inside this flex column.

# Product Card
flex-shrink-0 → Prevents the card from shrinking in a flex container.
w-full → Takes up full available width.
max-w-4xl → Maximum width capped at 64rem (~1024px).
p-4 → Padding inside (1rem = 16px).
border-4 border-blue-500 → 4px border, blue color.
rounded-lg → Rounded corners.
shadow-lg → Adds a large shadow (floating look).
flex flex-col → Inside layout is also a vertical flex column.

# Product Image
w-full → Image spans full card width.
h-[500px] → Fixed height of 500px.
object-cover → Crop image to cover without stretching.
rounded-lg → Rounded corners.
mb-4 → Bottom margin (1rem)

# Prodcut Infos
p-4 → Padding.
text-center → Centers text horizontally.
flex flex-col justify-center → Info stacked vertically, centered.
text-2xl font-bold → Big bold title.
truncate → If the name is too long, cut it with ....
text-lg → Slightly larger text for the price.

#Buy Button
mt-6 → Margin-top for spacing above button.
bg-blue-500 → Default blue background.
hover:bg-blue-600 → Darker blue on hover.
text-white → White text.
font-semibold → Medium-bold font.
py-3 px-8 → Padding: top/bottom 0.75rem, left/right 2rem.
rounded-lg → Rounded corners.
shadow-lg → Button shadow.
transition-colors duration-300 → Smooth hover color animation (300ms).
